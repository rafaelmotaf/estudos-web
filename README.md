# Full Stack Development 

Este repositório serve como documentação técnica e registro de evolução do meu estudo aprofundado em Desenvolvimento Web Moderno.

Aqui concentro os códigos, projetos práticos e desafios resolvidos ao longo de uma formação intensiva de mais de 100 horas, focada nas tecnologias mais exigidas pelo mercado atual. O objetivo deste repositório vai além de armazenar código; ele reflete minha transição da teoria para a prática na arquitetura de software escalável.

## Sobre o Projeto

O conteúdo deste repositório abrange desde a estruturação fundamental da web até a orquestração de containers e bancos de dados complexos. Cada diretório representa um passo na construção de uma stack robusta, onde o foco não é apenas fazer o código funcionar, mas entender o ciclo de vida da aplicação, boas práticas de escrita e otimização de performance.

## Tech Stack & Ferramentas

Minha abordagem de aprendizado é baseada no ecossistema JavaScript/TypeScript de ponta a ponta, integrando ferramentas de produtividade e controle de ambiente.

### Frontend Moderno
* **HTML5 & CSS3:** Domínio de layout responsivo, semântica e acessibilidade.
* **FlexBox & Grid:** Criação de layouts complexos e fluidos sem dependência de frameworks antigos.
* **Tailwind CSS:** Estilização utilitária para desenvolvimento ágil e consistente.
* **JavaScript (ES6+) & TypeScript:** Tipagem estática para códigos mais seguros e escaláveis.
* **React:** Construção de interfaces baseadas em componentes, gerenciamento de estado e hooks.
* **Next.js:** Renderização do lado do servidor (SSR) e geração de sites estáticos (SSG) para performance e SEO.

### Backend & API
* **Node.js:** Ambiente de execução para construção de servidores performáticos.
* **API REST:** Design e implementação de rotas, tratativa de erros e padronização de respostas.

### Banco de Dados & ORM
* **MySQL & MongoDB:** Modelagem de dados relacional e não-relacional.
* **Prisma:** Utilização de ORM moderno para migrações de banco de dados e type-safety nas queries.

### DevOps & Controle de Versão
* **Docker:** Containerização de aplicações para garantir paridade entre ambientes de desenvolvimento e produção.
* **GIT:** Controle de versão distribuído e fluxo de trabalho colaborativo.

## Estrutura de Aprendizado

Ao navegar pelas pastas, você notará uma progressão lógica:

1.  **Fundamentos:** Projetos estáticos focados em UI/UX e responsividade.
2.  **Lógica de Programação:** Algoritmos em JavaScript e transição para TypeScript.
3.  **Single Page Applications (SPA):** Aplicações reativas conectadas a APIs externas.
4.  **Ecossistema Full Stack:** Aplicações completas integrando Next.js, Banco de Dados e autenticação.
